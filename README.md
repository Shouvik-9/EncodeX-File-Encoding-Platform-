# 🧩 PROBLEM STATEMENT

## Users often face difficulties while generating QR codes due to:
- Complicated or cluttered interfaces  
- Lack of quick download and sharing options  
- No instant feedback or validation  
- Limited functionality in basic QR tools  

## This project aims to build a simple and interactive QR Code Generator that allows users to:
- Instantly generate QR codes from text or URLs  
- Download QR codes as images  
- Copy QR code links easily  
- Share QR codes directly from the browser  
- Enjoy a clean, modern, and responsive UI  

---

# 🧠 TECHNOLOGIES USED

- HTML5 – Structure of the application  
- CSS3 – Styling and animations  
- JavaScript (Vanilla JS) – Handles logic and interactions  
- QR Code API – Generates QR dynamically  

---

# 😮 WOW FACTORS

## ⚡ Instant QR Generation
- Real-time QR code creation using API  
- Supports both text and URL input  

## 📥 Download QR Code
- Save QR as PNG image  
- Uses Canvas for image export  

## 🔗 Copy QR Link
- Copy generated QR link to clipboard  
- Quick access for reuse  

## 📤 Share QR Code
- Native Web Share API support  
- Easy sharing across devices  

## ⌨️ Enter Key Support
- Generate QR instantly by pressing Enter  

## 🚨 Input Validation
- Prevents empty input  
- Shake animation for error feedback  

## 🎨 Modern UI Design
- Gradient animated title  
- Clean centered layout  
- Interactive hover effects
